Automation Program

Become an Automation Geek with the Tinkerstop Automation Program.
Learn different hardware and software aspects of Automation.
Indulge yourself in practical sessions, bonus sessions from experts and an industrial visit. 5 weeks of fun through learning and doing.

Register Here

What all you will be learning:

Hardware : Arduino, Raspberry Pi
Software : C++, Python, HTML, CSS, jQuery

Schedule :

Week 1:
It’s all about getting acquainted with Arduino and playing around by doing some cool automation projects with it. It would be a mix of learning electronics and programming concepts.
Goal : Make basic (yet cool) automation projects with Arduino and learn some programming.
→ Getting started with Arduino
→ Digital I/O, Analog I/O
→ Basic concepts of electronics and sensors
→ C++ in 2 days
Bonus lessons:
Internet 101 - Basics of the internet and how it works.

Week 2:
After learning to do some automation, we are now going to add a wireless communication layer to it to control it wirelessly from our mobile phone through SMS and bluetooth.
Goal : Learning to add wireless connectivity to hardware and control remotely.
→ Adding connectivity layer
→ GSM using SIM800
→ Bluetooth using HC-05
→ Serial Communication
Bonus Lesson:
A detailed insight into IoT.

Week 3:
A fastrack to front-end Web Development.
Goal : Learn to make graphic interfaces for user.
→ HTML
→ CSS
→ jQuery
Bonus Lesson:
Growing perspective on UI/UX.

Week 4:
A fastrack to back-end Web Development on Raspberry Pi and learn automation through the internet using WiFi.
Goal : Learn to create the backend for web interfaces.
→ Python
→ Flask
→ RGPIO
Bonus Lesson:
Getting to know about various tech-stacks of various web apps and automation products.

Week 5:
The last week is where we completely focus on prototyping of projects, an industrial tour and assessment.
Project phase :
→ Farm Automation
→ Real Time Monitoring of Data Centres
→ Home Automation
→ Security System
→ Fire Alarm System
→ Industrial Tour
→ Assessment and Certification.
Bonus Lesson : How to crack interviews from Industry expert.
